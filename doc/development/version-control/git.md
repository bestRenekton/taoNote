# 常用命令

## 新建

### 在当前目录新建一个 Git 代码库

`git init`

### 新建一个目录，将其初始化为 Git 代码库

`git init [project-name]`

### 下载一个项目和它的整个代码历史

`git clone [url]`

## 配置

# 显示当前的 Git 配置

`git config --list`

# 编辑 Git 配置文件

`git config -e [--global]`

# 设置提交代码时的用户信息

`git config [--global] user.name "[name]"`
`git config [--global] user.email "[email address]"`
